Question-1: What are the different ways to select an element in the DOM?

We can select an element in the dom by using getElementById(),getElementsByTagName(),getElementsByClassName(),querySelector() and querySelectorAll() methods.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

By using innerHTML we can get or set the html content including tags but by using innerText we can manupulate only the text inside the tag. textContent gets or sets all text including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a system where we can add event listener to parent element and this event listener works fine with the child elements

Question-4: What is event bubbling in the DOM?

Event bubbling means when an event occurs to an element, it's not only occurs to the element. the event propagates through it's parent and also it's grandparent.

Question-5: How do you create, add, and remove elements using JavaScript?

we can create an element using document.createElement(tagname). we can add by using appendChild() method. removing element can be happened by using remove() method. we can also remove elements by setting innerHTML to an empty string.
